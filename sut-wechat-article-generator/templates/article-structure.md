# 文章结构模板：黄金4+2

## 结构概览

```
┌─────────────────────────────────────┐
│  核心四章（必须）                    │
├─────────────────────────────────────┤
│  1. 钩子开头      100-150字         │
│  2. 核心价值      300-400字         │
│  3. 关键信息      400-500字         │
│  4. 行动指南      200-300字         │
├─────────────────────────────────────┤
│  可选两章（选择1-2个）               │
├─────────────────────────────────────┤
│  5. 深度解析 / 对比分析             │
│  6. 案例展示 / 未来展望             │
├─────────────────────────────────────┤
│  统一结尾       100-150字           │
└─────────────────────────────────────┘
```

---

## 核心四章（必须）

### 第1章：钩子开头（100-150字）

**目标：** 3秒内抓住注意力，制造继续阅读的欲望

**包含元素：**
- 痛点/悬念/数据/场景（选一种模式）
- 引出主题
- 承诺价值

**模板：**
```
[使用5种钩子模式之一生成]
```

**示例：**
```
每天写代码写到手软？重复的增删改查让你疲惫不堪？

最近我发现 Cursor AI，它能让编程效率提升3倍。

今天就来聊聊它是怎么做到的。
```

---

### 第2章：核心价值（300-400字）

**目标：** 快速让读者理解"这是什么"和"为什么重要"

**包含元素：**
1. **是什么（定义）** - 用通俗语言解释核心概念
2. **为什么重要（价值）** - 说明解决的问题或带来的价值
3. **核心卖点** - 2-3个最突出的特点

**结构：**
```
## 一、[主题]到底是什么？

[通俗定义，1-2段，每段不超过4行]

> 💡 **核心概念**：[用一句话总结核心概念]

## 二、为什么它值得关注？

[说明解决的问题，1-2段]

[说明核心价值，1-2段]

**它的核心特点在于：**
- [特点1]：[简短解释]
- [特点2]：[简短解释]
- [特点3]：[简短解释]
```

**示例：**
```
## 一、Cursor AI 到底是什么？

说白了，Cursor AI 就是一个"会写代码的编辑器"。

它基于 GPT-4 等大模型，能理解你的自然语言描述，
自动生成代码、解释代码逻辑、甚至帮你修复 bug。

> 💡 **核心概念**：Cursor AI = VS Code + AI 超级助手

## 二、为什么它值得关注？

传统编程方式，你需要：
- 手写每一行代码
- 自己查找 bug
- 重复写相似的逻辑

Cursor AI 让你只需描述需求，代码自动生成。

**它的核心特点在于：**
- **智能生成**：用自然语言描述需求，代码自动完成
- **上下文理解**：理解整个项目结构，生成相关代码
- **实时建议**：边写边提示，像有个经验丰富的助手在身边
```

---

### 第3章：关键信息（400-500字）

**目标：** 让读者理解"如何工作"和"有什么功能"

**包含元素：**
1. **工作原理** - 用生活化类比解释技术原理
2. **主要功能** - 详细列出核心功能
3. **技术亮点** - 独特的技术优势

**结构：**
```
## 三、它是如何工作的？

[先给生活化类比，1-2段]

[补充准确的技术解释，1-2段]

> 📖 **技术原理**：[简要说明核心技术]

## 四、核心功能一览

[功能1]
[功能2]
[功能3]

### 深度解析：[技术亮点]

[如果有独特的或复杂的技术点，可以展开说明]
```

**示例：**
```
## 三、它是如何工作的？

想象一下，你有个经验丰富的程序员助手坐在你旁边。

你告诉它："帮我写个登录功能"，它就开始写代码。

**1. 代码生成**

**2. 代码解释**

## SCQA 模型应用

在"核心价值"章节中，可以使用 SCQA 模型增强吸引力：

```
Situation（情境）：描述读者熟悉的现状
Complication（冲突）：指出存在的问题或痛点
Question（疑问）：引出核心问题
Answer（答案）：提供解决方案（主题内容）
```

**示例：**
```
作为开发者，你每天要写大量代码（S），
但重复的工作让你疲惫不堪，效率难以提升（C）。

有没有更好的方法？（Q）

Cursor AI 或许能给你答案——它能让编程效率提升3倍（A）。
```

## 可读性优化标准

### 段落控制
- ✅ 每段不超过4行
- ✅ 段落之间空一行
- ✅ 重要内容单独成段

### 视觉节奏
- ✅ 每300字必须有视觉元素（列表、引用、代码、图片）
- ✅ 避免连续3段纯文字
- ✅ 使用加粗强调关键词

### 信息密度
- ✅ 先结论后细节
- ✅ 先通俗后专业
- ✅ 先整体后局部
